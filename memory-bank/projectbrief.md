# Project Brief

## Tóm tắt dự án
ZaloCRM là hệ thống quản lý tập trung nhiều tài khoản Zalo cá nhân trên 1 giao diện web duy nhất. 

## Yêu cầu và mục tiêu cốt lõi
- **Quản lý nhiều Zalo:** Hỗ trợ đăng nhập bằng mã QR, tính năng tự động kết nối lại và duy trì phiên đăng nhập bảo mật.
- **Tương tác thời gian thực:** Chat real-time với đầy đủ tính năng gửi/nhận tin nhắn văn bản, hình ảnh, tệp tin, sticker và quản lý nhóm chat.
- **Quản lý khách hàng (CRM):** Xây dựng luồng pipeline theo dõi trạng thái khách hàng (Mới → Đã liên hệ → Quan tâm → Chuyển đổi → Mất).
- **Lên lịch và nhắc nhở:** Quản lý lịch hẹn với hệ thống thông báo/nhắc nhở tự động.
- **Báo cáo & Thống kê:** Cung cấp Dashboard trực quan (biểu đồ tin nhắn, KPI, trạng thái) và xuất báo cáo Excel theo thời gian.
- **Quản trị người dùng:** Phân quyền theo cấp độ (Owner / Admin / Member) để quản lý đội nhóm an toàn.
- **Tích hợp mở rộng:** Cung cấp REST API (xác thực bằng API Key) và Webhook cho việc tích hợp hệ thống bên ngoài.
- **Bảo vệ tài khoản:** Tích hợp cơ chế chống block Zalo (giới hạn 200 tin/ngày, cảnh báo tần suất).
